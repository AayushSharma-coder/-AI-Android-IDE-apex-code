# AI Android IDE
This is an Android IDE with Ai, plugin manager and version control  integration. The mobile code editor has all the basic features of an IDE with integrated AI and plugin store. There is an integrated AI chat box for coding and debugging help. You can also create branches of your main project just like GitHub. You can also download plugins just like in VScode.
